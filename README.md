# Nasayem Studio

**Visual Identity systems rooted in structural precision.**

Nasayem Studio is a design powerhouse dedicated to engineering visual equilibrium. This project is a modern, high-performance web experience built with React, GSAP, and Tailwind CSS, designed to showcase the "Physics of Visual Identity."

## Features

- **Dynamic Animations**: Smooth, GSAP-powered transitions and interactions.
- **Bi-lingual Support**: Seamless English and Arabic localization using `i18next`.
- **Structural Design**: A layout focused on hierarchy, clarity, and precision.
- **Optimized Performance**: Fast loading times and responsive design.

## Tech Stack

- **Framework**: [React](https://reactjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation**: [GSAP](https://greensock.com/gsap/)
- **Localization**: [react-i18next](https://react.i18next.com/)
- **Icons**: [Lucide React](https://lucide.dev/)

## Getting Started

### Prerequisites

- Node.js (v18+)
- npm

### Installation

1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:

```bash
npm run dev
```

### Build

Create a production-ready bundle:

```bash
npm run build
```
